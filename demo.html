Git Documentation

Git - This is a software/Tool, used for change control or version control.


Github - This one is a online service, used as host git repository/folder/source code.

Installation

  • Download and install git on your pc: https://git-scm.com
  • Then open command Line
    1. git --version
    2. git config --global user.name "user name"
    3. git config --global user.email "email"
    4. git config --list
    5. git config --global user.name "raju" // (we can change name)
    6. git config --global user.email "raju@gmail.com" // (we can change email)
    7. git config user.name
    8. git config user.email
